This is a project to display latest Capital Bikeshare map data by Carlos Torres. The data is checked every ten seconds, but if nothing has changed, it is not updated. They already have a map here, but I wanted to build my own. Right now it just shows current status, but I'm planning on adding more statistics in the future.

The project is was written in Python using Flask. The map is provided by Mapbox. The data is from the Capital Bikeshare API, which I store in an AWS RDS Postgres database.
